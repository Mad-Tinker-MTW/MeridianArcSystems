# Meridian Arc

The public website for **Meridian Arc Systems, LLC**.

> Meaningful Automation for Society  
> Because Your Time Matters.

## View locally

Install [Node.js](https://nodejs.org/), open a terminal in this folder, and run:

```bash
npm install
npm run dev
```

Open the address shown in the terminal, normally `http://localhost:3000`.

## Public website

Every update to the `main` branch is automatically built and published through
GitHub Pages:

`https://mad-tinker-mtw.github.io/MeridianArcSystems/`
